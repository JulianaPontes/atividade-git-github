## 👩🏾‍💻 Mulher homenageada

A mulher homenageada neste projeto é **Dorothy Vaughan**.

Dorothy Vaughan foi uma matemática e programadora norte-americana que teve papel fundamental na história da ciência, da tecnologia e da exploração espacial.

---

## 📚 Breve biografia

Dorothy Vaughan trabalhou no **NACA**, instituição que posteriormente se tornou a **NASA**. Ela atuou como uma das primeiras supervisoras negras da organização e liderou uma equipe de mulheres matemáticas conhecidas como **“computadores humanos”**.

Essas mulheres eram responsáveis por realizar cálculos essenciais para missões espaciais, em uma época marcada por fortes barreiras raciais e de gênero.

Com a chegada dos computadores eletrônicos, Dorothy Vaughan se destacou por aprender programação em **Fortran** e por ensinar outras mulheres a utilizarem essa nova tecnologia.

---

## 🏆 Principais conquistas

- Foi uma das primeiras supervisoras negras do NACA, que depois se tornou a NASA.
- Liderou uma equipe de mulheres matemáticas em um período de segregação racial e de gênero.
- Aprendeu programação em Fortran e ajudou outras mulheres a se prepararem para a computação eletrônica.
- Contribuiu para cálculos importantes relacionados ao programa espacial norte-americano.
- Tornou-se símbolo de liderança, inteligência, adaptação tecnológica e colaboração.

---

## 🌍 Curiosidades e impacto

Dorothy Vaughan é lembrada como uma referência de liderança, inteligência, colaboração e adaptação tecnológica.

Sua trajetória mostra a importância da diversidade na ciência e na tecnologia, além de inspirar mulheres, especialmente mulheres negras, a ocuparem espaços nas áreas de exatas, dados, computação e inovação.

A história de Dorothy Vaughan também ficou mais conhecida por meio do filme **“Estrelas Além do Tempo”**, que apresentou ao público a contribuição de mulheres negras para a NASA.

## 💻 O que aprendi sobre Git e GitHub

Durante esta atividade, aprendi que **Git** e **GitHub** são ferramentas importantes para versionamento de projetos e colaboração em tecnologia.

O Git permite acompanhar o histórico de alterações de um projeto, enquanto o GitHub permite publicar esse projeto online e colaborar com outras pessoas.

---

## 🧠 Comandos Git utilizados

### 1. `git init`

Inicializa um repositório Git dentro da pasta do projeto.

### 2. `git status`

Mostra o estado atual dos arquivos, indicando quais foram modificados, adicionados ou ainda não rastreados.

### 3. `git add`

Adiciona arquivos à área de preparação, deixando-os prontos para serem salvos em um commit.

### 4. `git commit -m "mensagem"`

Cria um registro das alterações feitas no projeto, com uma mensagem explicando o que foi alterado.

### 5. `git remote add origin`

Conecta o repositório local a um repositório remoto no GitHub.

### 6. `git push`

Envia os commits do computador para o repositório remoto no GitHub.

### 7. `git checkout -b nome-da-branch`

Cria uma nova branch e muda automaticamente para ela.

### 8. `git merge`

Junta as alterações de uma branch com outra.
